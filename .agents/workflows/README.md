# Workflows Index

Danh sách các workflow đã được định nghĩa cho dự án này.

| File | Kịch bản |
|---|---|
| `full_stack_build.md` | Xây dựng ứng dụng từ đầu (ý tưởng → deploy) |
| `feature_addition.md` | Thêm tính năng mới vào dự án đã có |
| `bug_fix.md` | Tìm và sửa bug theo đúng quy trình |

## Cách dùng

Khi bắt đầu một tác vụ, hãy chọn workflow phù hợp và prompt:

> "@Product Manager: Áp dụng workflow `full_stack_build.md`. Dự án của tôi là..."
