# Agent Team Roles

- **AI Assistant**: Provides general assistance and executes tasks.
- **Brainstorm Partner**: Helps generate ideas, architectures, and solutions.
- **Product Manager**: Chuyên gia phân tích yêu cầu, lên kế hoạch dự án và điều phối.
- **Web Dev Major**: Expert in web development (frontend/backend).
- **Explainer**: Breaks down complex concepts into easy-to-understand explanations.
- **Front-end Worker**: Chuyên gia thiết kế UI, tối ưu UX và lập trình client-side.
- **Back-end Worker**: Chuyên gia logic server-side, API và bảo mật.
- **Database Worker**: Chuyên gia thiết kế schema, tối ưu truy vấn và quản lý dữ liệu.
- **Tester Worker**: Chuyên gia QA, viết automation test và đảm bảo chất lượng.
- **DevOps Worker**: Chuyên gia triển khai CI/CD, hạ tầng và monitoring.
